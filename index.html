About

About the Project

This project is an AI-based touchless Human–Computer Interaction (HCI) system. It enables users to control a computer using head movements and voice commands.

Objective

  • Provide an accessible computing solution for users with physical disabilities.
  • Eliminate dependency on traditional input devices like mouse and keyboard.

Technologies Used

Python OpenCV MediaPipe Vosk PyAutoGUI Keyboard

Key Concepts

Computer Vision
Facial Landmark Detection
Offline Speech Recognition
Guide

Step-by-Step Guide

Follow these steps to get started with HandFreez.

1

Start the Application

Run the software on your computer.

2

Position Yourself

Sit in front of the camera with your face clearly visible.

3

Control Cursor with Head Movement

Move your head to control the cursor on your screen.

4

Switch Modes Using Head Tilt

Tilt head right → Mouse Mode
Tilt head left → Keyboard Mode
5

Use Voice Commands

Speak commands to perform actions:

"so click" "so right click" "so double click" "select all" "copy" "paste"
6

Typing with Voice

Speak normally to type text. Words are automatically entered as you speak.

7

Exit the System

Say "exit system" to close the application.
